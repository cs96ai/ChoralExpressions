# Choral Expressions

A marketing and fundraising website for **Choral Expressions**, a 33-member mixed-voice choral ensemble based in Toronto, Ontario.

## About

Choral Expressions is dedicated to classical mastery and soulful celebration. Our repertoire spans centuries—from Renaissance motets to contemporary gospel—united by a commitment to musical excellence and spiritual depth.

## Features

- **Hero Section**: Full-viewport dramatic presentation with animated elements
- **Event Services**: Hire us for Black History Month, Christmas, Easter, weddings, and more
- **Donation System**: Support musical education and repertoire acquisition
- **Booking Form**: Easy event inquiry submission
- **Responsive Design**: Beautiful on all devices

## Tech Stack

- **Framework**: .NET 8+ Blazor Server
- **Styling**: Bootstrap 5 with custom "Majestic & Soulful" theme
- **Icons**: Bootstrap Icons
- **Fonts**: Playfair Display (headers) + Inter (body)

## Design Theme

"Majestic & Soulful" - A premium, high-end aesthetic featuring:
- Deep Burgundy backgrounds (#1a0505, #2b1111)
- Gold/Bronze accents (#d4af37)
- Elegant serif typography
- Flashy hover effects with gold glow

## Getting Started

### Prerequisites
- .NET 8 SDK or later

### Run Locally
```bash
cd ChoralExpressions/ChoralExpressions
dotnet restore
dotnet run
```

Navigate to `https://localhost:5001` or `http://localhost:5000`
